LEGO and Open Source

What is LEGO?

  • The LEGO Group (TLG) is a worldwide group of companies that collectively is one of the biggest names in the toy world
  • LEGO bricks are the popular medium introduced in the 1950's, although LEGO makes many other interesting things
  • The Adult Fan of LEGO (AFOL) Community is a large group of enthusiasts and hobbyists around the world

Highlights of where LEGO has been

(because a surprising amount of people don't know about things that have been going on for decades)
  • 1930's: Carpenter Ole Kirk Kristiansen begins selling wooden toys
  • 1940's: LEGO begins selling plastic toys
  • 1950's: plastic takes over, modern brick system introduced
  • 1960's: rise of specialized elements, gears, motors, themed sets, DUPLO

Highlights of where LEGO has been

  • 1970's: minifigures, regular themes, Technic
  • 1980's: long-running themes, 9V electronics system, educational line
  • 1990's-2000's: short-running themes, explosion of colors, widely-available robotics platforms, video games, licensed lines, action figure lines

What is open source?

  • Software
    • Open-source software (OSS) is computer software with its source code made available and licensed with a license in which the copyright holder provides the rights to study, change and distribute the software at no cost to anyone and for any purpose. (Wikipedia)
  • Philosophy
    • In production and development, open source as a philosophy promotes a) universal access via free license to a product's design or blueprint, and b) universal redistribution of that design or blueprint, including subsequent improvements to it by anyone. (Wikipedia)

First, let's clear up misconceptions:

  • The LEGO brick is NOT open source
    • Selling anything with the word "LEGO" on it that TLG did not make is a bad idea.
  • LEGO is not an open source company. TLG's lawyers are aggressive in defending their intellectual property.
    • For the purposes of this presentation, we will not be discussing the merits of their legal cases
  • The LEGO Group has made efforts towards using open source to benefit themselves, the AFOL community, and the educational community
  • LEGO has open sourced some software (generally for Mindstorms robotics kits)
  • The LEGO website has a fair play policy outlining what they consider to be appropriate legal practices for fan sites.

Software: LDraw

  • Gold standard for LEGO CAD
  • (Arguably) better than LEGO's officially introduced software (Creator, Digital Designer)
  • The official Digital Designer program has some support for LDraw's file formats (see http://wiki.ldraw.org/index.php?title=LEGO_Digital_Designer and http://wiki.ldraw.org/index.php?title=LDraw_to_LDD_conversion for more)
  • Used for documenting and sharing original creations, and for generating building instructions

Software: LDraw enhancements

  • MLCad - graphical user interface for LDraw http://mlcad.lm-software.com/ (closed source, but the most popular user interface for LDraw)
  • LPub - documentation publishing tool preferred for creating building instructions: http://lpub.sourceforge.net/
  • LeoCAD - Separate CAD program using same parts file format (and can export LDraw file formats): http://www.leocad.org/trac
  • LDraw-Sketchup-Plugin - tool to import LDraw files into SketchUp: https://github.com/dannystaple/LDraw-Sketchup-Plugin

Software: LDraw enhancements

(continued)

  • Bricksmith - LDraw GUI program for Mac: http://bricksmith.sourceforge.net/ (BSD license)
  • LDView - OpenGL rendering program for viewing LDraw models: http://ldview.sourceforge.net/
  • LSynth - program that synthesizes bendable parts for LDraw files: http://lsynth.sourceforge.net/
  • LSculpt - program that converts triange mesh 3D models into LDraw files: https://code.google.com/p/lsculpt/

Software: Brickstore

  • BrickLink (http://www.bricklink.com/) is the primary online marketplace for buying LEGO parts. Like eBay, it is primarily peer-to-peer, but also includes professional sellers. Parts can be found using an Amazon-like catalog feature.
  • Sourcing parts for a large project on BrickLink is actually rather difficult, since you will likely need to purchase from multiple sellers, and each seller's prices and quantity available vary.
  • Brickstore is a popular tool for trying to make this easier.
  • Programming community support for Brickstore is weak, but the tool is seen as extremely valuable - and in need of improvements - by the AFOL community.

Software: Bluebrick

  • Gold standard for LEGO train clubs planning group layouts
  • http://bluebrick.lswproject.com/
  • Replaces Windows (.NET-based) open source project TrackDraw http://sourceforge.net/projects/trackdraw/
  • ...which itself superceded the closed-source Track Designer program

Software: Mosaics

  • PicToBrick: http://www.pictobrick.de/de/download.shtml
  • GIMP (GNU Image Manipulation Program) - widely used, but also popular for making studs-out mosaics (where a flat baseplate is filled as if 1 x 1 bricks were pixels). http://www.gimp.org/

Software: YubNub

"A social command line for the web". YubNub is a faster way of searching websites, and includes commands for searches of LEGO-specific databases.
Hosted at http://yubnub.org/ and on github: https://github.com/JonathanAquino/yubnub

Software: MINDdroid

  • One of LEGO's official Android apps, MINDdroid is a remote control for the NXT
  • Includes some support for NXTs running unofficial firmware
  • Available on Google Play for Android 2.1 and up: https://play.google.com/store/apps/details?id=com.lego.minddroid&hl=en
  • Source is available on github under GPLv3: https://github.com/NXT/LEGO-MINDSTORMS-MINDdroid

Software: Mindstorms firmware

Unofficial open source projects for the NXT:
NXTGCC: http://nxtgcc.sourceforge.net/
Improved Firmware: http://nxt-firmware.ni.fr.eu.org/
Enhanced firmware and BricxCC (IDE for programmable bricks): http://bricxcc.sourceforge.net/
There are enough of these to keep track of since LEGO officially open-sourced the NXT firmware that there is an unofficial guide to these: http://www.mindstorms.rwth-aachen.de/trac/wiki/FirmwareVersions

Software: LeJOS

  • The Java virtual machine has been ported to the LEGO Mindstorms EV3, RCX and NXT platforms, under alternate firmware.
  • http://lejos.sourceforge.net/

Software: Linux RCX Tools

  • Officially, LEGO only supported programming the RCX from Windows or a Mac.
  • The results of the The LegoUSB Project ( http://legousb.sourceforge.net/index.shtml ) have made support for the LEGO Mindstorms IR tower officially part of the linux kernel since the 2.6.1 release in 2004.
  • http://www.mralligator.com/rcx/tools.html offers Send, Firmdl, and Librcx, which allow sending messages, downloading firmware, and exploring the ROM interface, respectively.
    • Firmdl's source code makes an appearance in the movie Iron Man - implying that both LEGO pieces and open source software played a role in the suit's development.

Software: Mindstorms EV3 runs Linux

EV3 (introduced 2013) is LEGO's replacement for the previous "NXT" Mindstorms system. It can use the NXT's sensors and motors, but unlike the NXT, runs full-blown Linux.

  • LEGO's official firmware (based on Angstrom) is on github: https://github.com/mindboards/ev3sources
  • ev3dev project puts a flavor of Debian on the EV3: https://github.com/mindboards/ev3dev
  • A Python library is available: https://github.com/hmml/ev3

Philosophy & Software: Brickipedia

  • The Brickimedia Association (http://brickimedia.org/) runs several wikis (based on MediaWiki) of LEGO-related information, all licensed Creative Commons Attribution-ShareAlike 3.0.
  • Brickimedia wikis include:
    • Brickipedia (a LEGO encyclopedia)
    • Customs (a repository for sharing your original creations)
    • LEGO Stories (fan fiction)
    • Ideas Wiki (a resource for the LEGO Ideas website)
    • LEGO Minifigures Online Wiki (a guide to the online LEGO MMO)
  • Brickimedia's source code is on github: https://github.com/Brickimedia

Philosophy: User Groups

  • LEGO User Groups are patterned after technical users' groups, which have been around since the 70's
  • LEGO User Groups and Linux User Groups regularly have conflicting acronyms and web addresses

Philosophy: photo sharing

  • Brickshelf
  • Flickr
  • MOCPages
  • Although the Fibblesnork backgrounds (a fixture of the online LEGO community since 1996) predate Creative Commons (started in 2001), the terms for using them are similar to a CC-BY-ND license: they are freely available for use on any website, provided that every page using one includes a link back to the page about them (including the various sizes and colors as well as a simple "Terms of Use" explaining that) http://www.lugnet.com/fibblesnork/lego/backgrounds/

Philosophy: open data (databases)

These sites are not truly open source, but are database-based resources popular in the LEGO community that accept some form of contributions from the community.
  • BrickLink (Amazon-like catalog-based marketplace and inventory service) http://www.bricklink.com/
  • BrickSet (variety of LEGO set info, collection tracking) http://brickset.com/
  • Rebrickable (service for determining if you own the parts to build various kits or models people have uploaded inventories for): http://rebrickable.com/
  • Peeron (set inventories and catalog & instruction scans) http://www.peeron.com/
  • LUGNET (older/often out-of-date site with news and references on sets, parts, LEGO User Groups, "cool sites", and more) http://www.lugnet.com/

Philosophy: open data (APIs)

None of the below are truly open source. All of these APIs expose resources built by the LEGO community to other developers.
  • Brick Owl has an API ( http://www.brickowl.com/api_docs )
  • BrickLink offers a store API ( http://api.bricklink.com/pages/api/welcome.page ) and catalog download ( http://www.bricklink.com/catalogDownload.asp )
  • BrickSet offers a web services API: http://brickset.com/tools/webservices
  • Rebrickable has an API ( http://rebrickable.com/api/ ) and cached data for download ( http://rebrickable.com/downloads )

Philosophy: Custom 3D-printed elements

  • You can use open hardware 3D printers and files from Thingiverse (http://www.thingiverse.com)
  • Most 3D printers use the same type of plastic used to make LEGO bricks, but cannot match LEGO's quality standards (although acetone vapor can make the appearance similar).
  • Purist LEGO hobbyists won't use these - if it's not made by LEGO, it's "cheating". Hackers are usually happy to use these. We can't agree on everything.
